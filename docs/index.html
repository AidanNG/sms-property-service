Property Lookup SMS Service

This SMS service, operated by Aidan Gow, allows users to receive real estate property information by texting an address to our toll-free number: +1 (833) 699-8604.

When a user sends an address, our system automatically looks up public property data and replies with basic information such as bedrooms, bathrooms, and estimated property value.

Opt-In & Consent

Users voluntarily initiate contact by texting the number above. By sending the first message, users provide consent to receive automated replies containing property information related to their request.

This service does not send unsolicited messages or marketing content. Replies are only sent in response to user-initiated requests.

Opt-Out Instructions

Users can stop receiving messages at any time by replying with STOP. This immediately ends message delivery.
